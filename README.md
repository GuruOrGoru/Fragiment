# Code Share

A Rails application for sharing code snippets with syntax highlighting, comments, likes, and more.

## Features

- User authentication with Devise
- Code snippets with syntax highlighting using Rouge
- MacBook-style code preview
- Like and comment on snippets
- Nested replies on comments
- Search and filter snippets
- Pagination
- Admin panel with Rails Admin
- Responsive UI with Tailwind CSS
- File uploads for images and videos

## Setup

1. Clone the repository
2. Install dependencies: `bundle install`
3. Set up the database: `rails db:create db:migrate db:seed`
4. Start the server: `bin/dev`

## Usage

- Sign up or log in
- Create new code snippets
- Browse and search snippets
- Like and comment on snippets
- Admins can access /admin for management

## Testing

Run tests with: `rails test`

## Deployment

For production, configure your database and secrets. Use PostgreSQL in production.
